    Pixas Engine

    Launch : Dec 2011

    Pixas is an ActionScript engine, which enables programmers to build isometric pixel Flash applications with pure ActionScript3.0. With Pixas, you can easily add isometric pixel elements like brick, cube, pyramid, slope and map layer to your applications.

    It is isometric graphic based, but not designed as an isometric game engine. I can't tell what it's exactly for, just dive into it and try to pixelate something.

    Pixas is open source with MIT licence. You can check out the source at:

    Github :

    https://github.com/rison/pixas
