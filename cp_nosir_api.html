    Get current browser's userAgent type

    Return [String] - ie, chrome, firefox, safari, opera, other

    alert(nosir.ua);

    Run Code

    nosir.ie

    Get if the browser is MSIE core based browser

    Return [Boolean] - if the browser is MSIE core based browser

    alert(nosir.ie);

    nosir.uo

    Get the url query object and the hash value

    Return [Object] - the url query object

    //url.html?from=web#top

    nosir.defer( [array] )

    Defer excuting functions after all of the elements in DOM loading over

    array [Array] - An array for all functions to be defer loaded

    defer([doJobA, doJobB]);

    nosir.$( [id] )

    Select the document element by id

    id [String] - id of the element

    Return [Object] - the reference of the element

    var logo = nosir.$("Logo");

    nosir.css( [container] , [style] );

    Define multiple css properties at once by className or style object

    container [String / Object] - element name / reference

    style [String / Object] - class selector's name / object

    .style-test-css {font-size:12px, background-color:#FFF} (in css file);
    nosir.css("Testing_css", "style-test-css");

    nosir.css(nosir.$("Testing_css"), {fontSize:"12px", backgroundColor:"#FFF"});

    Run CodeReset

    This is the container for testing css

    nosir.inner( [container] , [html] )

    Set the inner Html contend for container

    container [String / Object] - element name / reference

    html [String] - html content

    nosir.inner("Logo", "This is the text replace the logo");

    nosir.outer( [container] , [html] )

    Set the outer Html contend for container

    container [String / Object] - element name / reference

    html [String] - html content

    nosir.outer("Logo", "This is the text replace the logo node");

    nosir.append( [container] , [html] )

    Append the Html contend for container

    container [String / Object] - element name / reference

    html [String] - html content

    nosir.append("Logo", "This is the text append after logo");

    Do not use append function within large loop due to the efficiency problem, instead of, generate the html string first then inner it into the container.

    nosir.swf( [container] , [id] , [src] , [width] , [height] , [wmode] , [flashvars] )

    Insert swf into container

    container [String / Object] - element name / reference

    id [String] - swf id. Every swf in the DOM should have the only id

    src [String] - swf address

    width [String] - width + "px" / percent + "%"

    height [String] - height + "px" / percent + "%"

    wmode [String] - opaque / transparent / window / direct

    flashvars [String] - init vars for flash

    Return [String] - the html code of the flash block

    nosir.swf("Testing_swf", "Testing_swf", "http://www.youtube.com/v/7oBQnIumBRY", "100%", "100%", "opaque", "version=3&from=risonhuang.com");

    Run CodeReset

    This is the container for testing swf

    nosir.$swf( [swf_id] )

    Select the swf element by id

    swf_id [String] - id of swf that has been inserted into dom by nosir.swf()

    Return [Object] - the reference of the swf element

    The swf reference returned could be used for calling ActionScript functions, like this:

    var someSwf = nosir.$swf("SomeSwf");
    swf.callByJavaScript(params);

    nosir.addEvent("Testing_addEvent", "mouseout", onTestingAEMouseOut);
    function onTestingAEMouseOver() {

    nosir.fade("Testing_addEvent", 0, 0.2);

    function onTestingAEMouseOver() {

    nosir.fade("Testing_addEvent", 100, 0.2);

    }

    nosir.removeEvent( [container] , [type] , [handler] )

    Remove standard DOM event listener from handler function

    container [String / Object] - element name / reference

    type [String] - event type

    handle [Function] - event handler function

    nosir.getEvent()

    Get standard DOM event reference

    Return [Event] - triggered event reference

    nosir.addEvent("Testing_getEvent", "click", onTestingGEMouseOver);
    function onTestingGEMouseOver() {

    var evt = nosir.getEvent();

    alert("container id : " + evt.target.id);

    }
