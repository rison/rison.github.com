    NoSir Library

    Launch : Sep 2009

    Nosir Library is a lightweight open-source JavaScript library. It helps to simplify the code of browser compatibility, Event handling, DOM manipulating and CSS selecting. It is brief and efficient, but totally follow the way you normally write JavaScript. Like this :

    nosir.fade( nosir.$("Logo") , 0 , 0.3 );

    Run CodeReset

    And all of the objects and functions are enclosed in the certain Object - nosir, so it works totally fine with any other JavaScript engine or library.

    You can check out the source at:

    Github :

    https://github.com/rison/nosir

    Google Code :

    http://code.google.com/p/nosir

    Features

    small (5 KB, Minified)

    cross-browser (Chrome, IE 6.0+, FF, Safari)

    works well with any individual code and libraries like jQuery

    swf object easily manipulating

    high efficience (without any DOM contamination)
