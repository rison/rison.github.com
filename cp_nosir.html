    NoSir Library

    Launch : Sep 2009

    Nosir Library is a lightweight JavaScript library which was widely used in my previous team, and now it is open-sourced. It helps to simplify the code of browser compatibility, Event handling, DOM manipulating and CSS selecting. It is brief and efficient, but totally follow the way you normally write JavaScript. Like this :

    nosir.fade( nosir.$("Logo") , 0 , 0.3 );

    Run CodeReset

    With module pattern design thoughts, all of the objects and functions are enclosed in a certain Object - nosir, so it works totally fine with any other JavaScript engine or library.

    Features

    small (5 KB, Minified)

    cross-browser (Chrome, IE 6.0+, FF, Safari)

    works well with any individual code and libraries like jQuery

    swf object easily manipulating

    high efficience (without any DOM contamination)
