    Photo To Video Platform

    2006 - 2012 http://photo.56.com/mv/make/ errors might pop up when accessing from out of China due to CDN limit problem

    This platform enables users to create MV by uploading photos and music, selecting effects, entering subtitle. The creating page was implemented by JavaScript and HTML/CSS. The final application was actually a Flash shell, it loads the XML including all of the assets data, then displays with animations and effects. (Photos from Dribbble.com)

    Features

    Photo drag sorting

    Photo compressed in client-side before uploading

    Multiple file upload
